部屋の好みをMBTIのように診断するアプリ「住まいのMBTI（住まいタイプ診断）」を開発。若者が住まいについて主体的に考える機会が少なく、妥協して部屋を選びがちであることや、自身の好みを言語化できていないという課題に対し、直感的な質問に答えるだけで深層心理にある住まいの嗜好を可視化し、理想の暮らしを具体的にイメージできる体験を設計した。
曖昧だった「好き」をパラメータ化する独自手法による定量的診断に加え、ChatGPTを開発パートナーとして活用するバイブコーディングを実践し、非エンジニアでもプロダクト開発に取り組める可能性を示した。今後は不動産領域への応用や、ライフスタイル全般への診断領域拡張を見据えている。
